          # Planung:
          - Merkmale der Planung
            - Zukunftsbezogenheit
              -  Problem Unvollkommene Information -> Fehlplanung
            - Systematik
              - Rational / Methodisch
            - Prozesscharakter
              - Auch wiederholend ?
            - Gestaltungscharakter
              - Gestaltung der Zukunft durch den Plan
            - Informationscharakter
              -  Information sammeln und verarbeiten und speichern
          
          "Wenn man plant versucht man ein Lösung für ein zukünftiges Problem anhand aktueller Informationen"- Ich
          
          "Planung ist die Zeitraum bezogene, geistige Vorwegname, von zukünftigen Ereignissen." - Der Lehrer
          
          
          - Kategorien der Planung
            -  Planungsgegenstände
          
          
          - Bezugszeiträume
            - kurz-, mittel-, lang- fristig
            - < 1 Jahr, 1-3 Jahre, > 3 Jahre
            -  stark abhängig von Branche
          
          -  Planungshierarchie
            -  Top - Strategisch
            - Middle - Taktisch
            - Lower - Operativ
          
          - Leitungshierarchie
            -  Ganzes Unternehmen bis runter zur Stelle (kleinste Organisationseinheit)
          
          
          - Funktionsbereiche
            - Beschaffung
            - Produktion
            - Absatz
            - Finanzierung
          
          - Koordination
            - top -> down  (retrograd)
              - Ausgangspunkt ist das Ergebnis -> Zwischenziele / Teilpläne -> Einzelpläne / Detailpläne
              - Nachteile: Vernachlässigen von Details?, Wiederstand von Unten, Vision kommt nicht richtig unten an
            - bottom -> up (progessive)
              - Detailplanung -> Umfassende Planung
              - Detailverliebtheit, Enormer Koordinationsaufwand, Fachwissen fehlt manchmal
            - Sekulär
              - Unten -> Oben -> Unter
          
          
          - Phasen der Planungsprozesses 
            - Zielbildung (später mehr)
            - Problemfeststellung
            - Alternativensuche
            - Prognose
            - Alternativenbewertung und Entscheidung
